# kernel density estimation

> [!summary] Quick definition
> Kernel density estimation estimates a smooth density by averaging kernel functions centered at observed data points.

## When it matters

KDE is useful for exploratory density plots and nonparametric diagnostics. Bandwidth choice controls smoothness and can strongly affect visual conclusions.
