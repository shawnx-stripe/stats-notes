# transportability

> [!summary] Quick definition
> Transportability concerns whether a causal effect learned in one setting can be validly applied to another target setting.

## When it matters

Transport needs clear target populations, effect modifiers, and differences in treatment implementation or context. Reweighting can help only when the target support is represented in the source data.
