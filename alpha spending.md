# alpha spending

> [!summary] Quick definition
> Alpha spending allocates a fixed type I error budget across interim analyses in a group-sequential design.

## When it matters

It lets analysts stop early while preserving the planned false positive rate. The spending function and information times should be pre-specified before monitoring begins.
