# falsification tests

> [!summary] Quick definition
> Falsification tests probe implications that should be false if the design assumptions are credible.

## When it matters

Examples include placebo outcomes, pre-treatment effects, fake cutoffs, and unaffected groups. Passing them does not prove identification, but failures clarify which assumption is under strain.
