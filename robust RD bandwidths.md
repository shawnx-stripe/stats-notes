# robust RD bandwidths

> [!summary] Quick definition
> Robust RD bandwidths are bandwidth choices designed for bias-corrected RD estimation and valid coverage.

## When it matters

MSE-optimal bandwidths can undercover if bias is ignored. Robust RD workflows report conventional and robust estimates, bandwidths on both sides, and sensitivity over reasonable windows.
