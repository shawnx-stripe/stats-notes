# identifiability

> [!summary] Quick definition
> A parameter is identifiable if different parameter values imply different observable distributions under the model.

## When it matters

Without identifiability, more data cannot pin down the target. Check rank conditions, normalization choices, support, and whether latent-variable labels or structural parameters are uniquely determined.
