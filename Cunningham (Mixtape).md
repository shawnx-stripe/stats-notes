# Cunningham (Mixtape)

> [!summary]
> Scott Cunningham. *Causal Inference: The Mixtape* (2021) — accessible textbook covering potential outcomes, matching, IV, RDD, DiD, and synthetic control with code examples in R and Stata.

## Key contributions

The Mixtape emphasizes intuition over formalism, using narrative examples (abortion legalization, minimum wage) to motivate causal inference methods. Includes detailed code walkthroughs for practitioners learning DiD, synthetic control, and RDD. The book's conversational tone makes technical concepts accessible to applied researchers without PhD-level preparation. Particularly strong on recent developments in staggered DiD (Callaway–Sant'Anna, Sun–Abraham) and synthetic control extensions.

> [!note]
> Free online version available at mixtape.scunning.com. Code repository includes complete replication files in R, Stata, and Python for all empirical examples.
