# dose–response function

> [!summary] Quick definition
> A dose-response function maps treatment intensity to expected potential outcomes.

## When it matters

It is the target for continuous or multi-valued treatments. Identification requires overlap across doses and assumptions such as generalized unconfoundedness or valid instruments.
