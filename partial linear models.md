# partial linear models

> [!summary] Quick definition
> A partial linear model combines a linear target component with a flexible nuisance function.

## When it matters

It is central to double machine learning: estimate nuisance functions flexibly, residualize treatment and outcome, then estimate the low-dimensional causal parameter.
