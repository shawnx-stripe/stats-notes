# matrix completion for panels

> [!summary] Quick definition
> Matrix completion for panels imputes missing untreated potential outcomes using low-rank structure in unit-by-time outcomes.

## When it matters

It generalizes synthetic-control and interactive fixed-effect ideas to multiple treated units and staggered missing counterfactuals. Credibility depends on pre-treatment fit and low-rank assumptions.
