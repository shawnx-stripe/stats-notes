# robust regression

> [!summary] Quick definition
> Robust regression estimates relationships using loss functions or weighting schemes less sensitive to outliers than least squares.

## When it matters

It is useful when heavy tails or influential observations dominate OLS. It changes the target estimand, so report the loss function and compare with OLS or quantile regression.
