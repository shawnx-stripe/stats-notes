# auxiliary variables

> [!summary] Quick definition
> Auxiliary variables are not central analysis variables but help predict missingness or missing values.

## When it matters

Including strong auxiliary variables can make MAR more credible and improve multiple imputation or weighting precision. They should be pre-treatment or otherwise safe to condition on for the estimand.
