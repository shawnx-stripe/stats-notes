# model misspecification

> [!summary] Quick definition
> Model misspecification occurs when the fitted statistical model omits or distorts features of the data-generating process relevant to the estimand.

## When it matters

It can bias estimates, invalidate standard errors, or change interpretation. Use diagnostics, flexible nuisance models, robustness checks, and design-based identification where possible.
