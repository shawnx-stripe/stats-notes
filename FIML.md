# FIML

> [!summary] Quick definition
> Full Information Maximum Likelihood estimates model parameters using all observed components of incomplete records under a likelihood model.

## When it matters

FIML is common in structural equation models and longitudinal models with incomplete outcomes. It is efficient when the model is correct and missingness is ignorable, usually MAR given included variables.
