# announcement vs implementation

> [!summary] Quick definition
> Announcement and implementation dates can define different treatment onsets when agents react before a policy takes effect.

## When it matters

Using only the implementation date can miss anticipatory behavior; using announcement dates can misclassify exposure if implementation is uncertain. Report both when behavior plausibly changes at announcement.
