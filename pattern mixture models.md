# pattern mixture models

> [!summary] Quick definition
> Pattern mixture models stratify the outcome distribution by missingness pattern and combine those pattern-specific distributions.

## When it matters

They are useful for sensitivity analysis when MAR is questionable. Analysts specify or vary assumptions about unobserved outcomes in each missingness pattern, then report how conclusions change.
