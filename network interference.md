# network interference

> [!summary] Quick definition
> Network interference occurs when one unit's treatment affects outcomes of connected units in a graph.

## When it matters

It invalidates individual-level no-spillover assumptions. Designs need exposure mappings, cluster randomization, graph cuts, or saturation variation to estimate direct and spillover effects.
