# identification region

> [!summary] Quick definition
> An identification region is the set of parameter values consistent with the data and maintained assumptions.

## When it matters

When assumptions do not point-identify an effect, reporting the identified set is more honest than a single estimate. Bounds can then be paired with sampling uncertainty for inference.
