# Bayesian bootstrap

> [!summary] Quick definition
> The Bayesian bootstrap places random Dirichlet weights on observed data points instead of resampling rows.

## When it matters

It provides a lightweight posterior-like uncertainty calculation for functionals of the empirical distribution. It is useful for weighted means and smooth summaries, but it does not replace a full likelihood model.
