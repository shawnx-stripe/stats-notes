# buffer zone design

> [!summary] Quick definition
> Buffer zone designs separate treated and control units with excluded geographic or network margins to reduce spillovers.

## When it matters

Buffers can improve internal validity but reduce sample size and change the target population. Report buffer rules and sensitivity to alternative widths.
