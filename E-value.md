# E-value

> [!summary] Quick definition
> An E-value summarizes the minimum unmeasured-confounding strength needed to explain away an observed association.

## When it matters

It is a sensitivity-analysis summary, not a fix for confounding. Interpret it alongside plausible confounder strengths and study design.
