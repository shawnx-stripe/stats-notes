# Always Valid Inference (AVI)

> [!summary] Quick definition
> Always-valid inference gives tests or intervals that remain valid no matter when data collection stops.

## When it matters

AVI is designed for peeking, optional stopping, and continuous experiment monitoring. It requires procedures built for that setting; ordinary fixed-sample p-values are not always-valid.
